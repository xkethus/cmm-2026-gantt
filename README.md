# CMM 2026 · Gantt interactivo y saturación

Visualización interactiva del programa 2026 del Centro Multimedia: diagrama de Gantt
de las actividades por área y mapa de calor de saturación diaria (actividades
simultáneas), con filtros y tooltips.

## Uso

Es un sitio estático de un solo archivo. Abre `index.html` en el navegador, o
despliégalo en cualquier hosting estático (Vercel, Netlix, GitHub Pages).

## Despliegue en Vercel

No requiere configuración: Vercel detecta el `index.html` en la raíz y lo publica
como sitio estático.
